AI-Powered Medical

Chest X-ray analysis using state-of-the-art vision transformer architecture

Built an end-to-end deep learning system to classify pneumonia from chest X-ray images using the Swin Transformer architecture — a cutting-edge vision model that outperforms traditional CNNs on medical imaging tasks. The project covered everything from dataset preprocessing to production-ready model evaluation.

  • Implemented Swin Transformer in Python using TensorFlow and Keras, leveraging transfer learning to maximise accuracy on a limited medical dataset
  • Designed a robust preprocessing pipeline — normalisation, flipping, rotation, zoom — to improve model generalisation and reduce overfitting
  • Benchmarked multiple architectures, iterating on hyperparameters to achieve optimal classification performance
  • Conducted systematic model evaluation using standard metrics, producing a publication-ready research paper
  • Co-authored and presented the findings at an IEEE conference to an international academic audience
